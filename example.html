COSCUP 2010 版面範例

請看原始碼了解各處的文字要從哪裡填入。

內文原則

  • 乾淨 HTML，維持格式統一，除非例外（例如，把 ordered list 的標號改成字母）嚴禁 inline style 與 style block。
  • 請善用 960.gs 所定義的格線寬度。
  • 請用 HTML 模式編輯內文，TinyMCE 不太乾淨。若有困難請洽文字編輯。

安裝

要調 style 這個檔案應該夠了。想要玩的話或是改 HTML 輸出的話才需要裝 WordPress 來執行這個佈景。

  1. 安裝 WordPress
  2. 在 /wp-admin/options-permalink.php 啟用固定網址。
  3. 在 /wp-admin/edit-pages.php 建立兩個分頁，共三個，網址分別是 /root、/zh-tw、/en。
  4. 在 /wp-admin/options-reading.php 把 /root 那頁設定為首頁。該頁面的內容不會被讀到，會自動跳到 /zh-tw
  5. 建立其他需要的頁面，指定到中文首頁或是英文首頁底下。
  6. 到 /wp-admin/widgets.php 對各網頁模組區塊填入需要的內容。中英有別，請愛用文字模組。
